Introduction

Javascript (Not to be confused with Java (programming language), Javanese script, or ECMAScript) often abbreviated as JS, is a programming language and core technology of the Web, alongside HTML and CSS. 99% of websites use JavaScript on the client side for webpage behavior.

All major web browsers have a built-in JavaScript engine that executes the code on the user's device. Web browsers have a dedicated JavaScript engine that executes the client code. These engines are also utilized in some servers and a variety of apps. The most popular runtime system for non-browser usage is Node.js.

The use of JavaScript has expanded beyond its web browser roots. JavaScript engines are now embedded in a variety of other software systems, both for server-side website deployments and non-browser applications. JavaScript has recently begun to appear in some embedded systems, usually by leveraging Node.js.

Variadic function demonstration

function sum() {
    let x = 0;

    for (let i = 0; i < arguments.length; ++i)
    x += arguments[i];
            
    return x;
 }
            
sum(1, 2); // Returns 3
sum(1, 2, 3); // Returns 6
